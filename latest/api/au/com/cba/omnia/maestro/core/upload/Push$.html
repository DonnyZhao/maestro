  21. def push(src: Data, hdfsLandingDir: String, archiveDir: String, hdfsArchiveDir: String): Hdfs[Copied]

    Copy the file to HDFS and archive it.

    Copy the file to HDFS and archive it.

    If nothing goes wrong, this methods will:

    • copy the file to Hdfs
    • create an "ingestion complete" flag on Hdfs
    • archive the file

    This method will fail if a duplicate of the file or it's ingestion complete or processed flags already exist in HDFS, or if an archived copy of the file already exists locally or on HDFS.

    This method is not atomic. It assumes no other process is touching the source and multiple destination files while this method runs.
