    Loads the supplied text files and converts them to the specified thrift struct.

    The operations performed are:

    1. Append a time field to each line using the provided time source.
    2. Split each line into columns/fields using the provided delimiter.
    3. Apply the provided filter to each list of fields.
    4. Clean each field using the provided cleaner.

    6. Validate each struct.

  16. def loadWithKey[A <: ThriftStruct](delimiter: String, sources: List[String], errors: String, timeSource: TimeSource, clean: Clean, validator: Validator[A], filter: RowFilter, none: String)(implicit arg0: Decode[A], arg1: Tag[A], arg2: Manifest[A]): TypedPipe[A]

    Same as load but also appends a unique key to each line.

    Same as load but also appends a unique key to each line. The last field of A needs to be set aside to receive the key as type string.

    The key is generated by:

    1. Creating a random 4 byte seed for each load
    2. For each file hashing the seed and the path and then taking the last 8 bytes.
    3. Hashing each line and taking 12 bytes, taking the map task number (current slice number) and the offset into the file.
    4. Concatenating the file hash from 2, the slice number, byte offset and line hash

    This produces a 256 bit key.
